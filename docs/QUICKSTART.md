# Quick Start — AFK Journey Trainer

## Before you begin

- Backup your account / save data before enabling heavy modifications.
- Extract the trainer **outside** Steam game directories.
- Run the game in windowed mode first if you experience overlay issues.

## Installation

1. Open **Releases** and download the latest Windows archive.
2. Extract to a folder you control (e.g. `C:\Tools\afk-journey-trainer`).
3. Launch **AFK Journey** through Steam and wait until the main menu loads.
4. Run the main binary from the release folder **as administrator**.
5. Alt-tab back to the game — the trainer attaches via pattern scanning.

## Using the trainer

| Step | Action |
|---|---|
| Open menu | Press **`Insert`** in-game |
| Toggle feature | Press **`Num 1`** – **`Num 9`** or use overlay checkboxes |
| Adjust speed | Auto Battle menu → set multiplier (1x–10x) |
| Close overlay | Press **`Insert`** again |

## Recommended first session

1. **`Num 1`** — Unlimited Gems & Gold → verify currency in shop/gacha UI  
2. **`Num 3`** — All Heroes & Stars → check hero roster  
3. **`Num 2`** — Instant Max Hero Level → confirm ascension caps  
4. **`Num 4`** — Auto Battle & Speed → start a stage, set 2x–3x initially  
5. **`Num 5`** — Equipment & Artifact Max → open gear screen  

Enable **`Num 7`** (Instant Campaign) and **`Num 8`** (ESP / One Hit Kill) only when needed — they have the highest impact on battle flow.

## Tips

- Restart the trainer after every major game patch — signatures refresh on attach.
- Start with lower speed multipliers (2x–3x) before jumping to 10x.
- Use **`Num 9`** before closing the game to boost simulated offline rewards.
- Enable verbose logging in the overlay **Settings** tab when reporting issues.

## Troubleshooting

| Issue | Fix |
|---|---|
| Trainer does not attach | Launch game first, then run as administrator |
| Features have no effect | Wait for main menu, re-press hotkey, reopen overlay |
| Game crash in battle | Disable Num 4 high speed and Num 8 one-hit kill |
| Overlay not visible | Windowed/borderless mode · disable fullscreen exclusive |
| SmartScreen block | More info → Run anyway, or build from source |
| Patch broke features | Download latest release — patterns updated per version |

## After a game update

1. Download the newest trainer release  
2. Close both game and trainer completely  
3. Launch AFK Journey → wait for main menu  
4. Run trainer as administrator — automatic signature scan runs on attach
